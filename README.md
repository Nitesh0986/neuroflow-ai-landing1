# NeuroFlow-AI-Landing
A premium AI-powered SaaS landing page built with React and Tailwind CSS, featuring dynamic multi-currency pricing, responsive Bento-to-Accordion layout, optimized state isolation, semantic HTML, and high-performance animations for a frontend engineering challenge.
# 🚀 NeuroFlow AI Landing

A premium, responsive SaaS landing page for an AI-driven data automation platform built as part of a frontend engineering speed challenge.

## ✨ Features

* 🎯 Modern Hero Section
* 🧩 Responsive Bento Grid
* 📱 Automatic Bento → Accordion transition on mobile
* 💾 State persistence across viewport changes
* 💰 Dynamic Pricing Engine

  * Monthly / Annual billing
  * INR (₹), USD ($), EUR (€)
  * Matrix-driven pricing calculation
* ⚡ Optimized state isolation to minimize unnecessary re-renders
* 🎨 Premium UI with smooth CSS animations
* 🌐 Fully Responsive Design
* ♿ Semantic HTML & Accessibility
* 🔍 SEO Optimized

## 🚀 Getting Started

### Clone Repository

```bash
git clone <repository-url>
```

### Navigate to Project

```bash
cd NeuroFlow-AI-Landing
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
